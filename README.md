### VocaBoost
A vocabulary learning project.
# VocaBoost
# Overview

VocaBoost is a smart vocabulary enhancement application designed to help users learn, revise, and retain words effectively. The platform focuses on consistent practice through revision systems, interactive word cards, learning history, and personalized progress tracking.

Whether you are preparing for competitive exams, improving communication skills, or simply expanding your vocabulary, VocaBoost creates a structured and engaging learning experience.


## Features

# 1. Vocabulary Learning
Learn new English words daily
Understand meanings, synonyms, antonyms, and usage
Improve pronunciation and sentence formation

# 2. Revision System
Smart revision reminders
Spaced repetition-based learning approach
Helps users retain words for long-term memory

# 3. Word Cards
Interactive flashcard-style learning
Front side: Word
Back side: Meaning, example sentence, pronunciation, synonyms
Swipe or tap navigation for quick revision

# 4. Profile Dashboard
Personalized user profile
Track vocabulary growth
Monitor streaks and learning consistency
Performance analytics and progress reports

# 5. Learning History
View previously learned words
Access revision records
Check mastered and weak vocabulary areas

# 6. Favorites & Saved Words
Save important words
Create custom revision lists
Bookmark difficult vocabulary
